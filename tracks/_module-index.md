## How this file gets updated

- **A track file references a new module:** add a row here and link from the track file using the canonical URL above.
- **A title or URL changes upstream:** update this file *first*, then update every track-file occurrence.
- **A module is retired:** mark `❌ retired` here, leave the row for history, and update track files to use a replacement.
- **Quarterly:** maintainer or contributor walks every URL above, updates the `Last verified` column, files issues for any 404s.

This file is the single source of truth. If a track file disagrees with this index, the index wins.
